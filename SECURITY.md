# Security Policy

The developers of ShiftLedger take the security of your data seriously. 

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the maintainer yousef.ehab.k@gmail.com or open a private issue if available.

### All data is Local
Please note that ShiftLedger stores all data locally in your browser's `localStorage`. No data is ever transmitted to a server. This design is intended to provide maximum privacy and security by default.

## Supported Versions

Only the latest version of this Progressive Web App (PWA) is supported. Users are encouraged to refresh their browsers to ensure they are running the latest version with the newest security updates.
